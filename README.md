### Curriculum Overview

* **Workbook 1:** Introduction to Excel covering **data types** and **data analysis**.
* **Workbook 2:** Intro to Tableau and Power BI, focusing on **importing** and **analysing** data.
* **Workbook 3 (Week 3):** Database design and **SQL**.
* **Workbook 5 (Week 5):** Fundamentals of **Azure**.
* **Workbook 6 (Week 6):** **Python** programming and data visualisation libraries.
